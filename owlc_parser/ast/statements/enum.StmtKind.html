Variants

Block(Block)

Wrapper around Block.

Fn

Represents a function declaration (with definition if not extern). e.g. fn f(x) { return 3; } Field body.kind should always be variant StmtKind::Block. If field body is None, function is an extern function.

Grammar

fn = ( fn proto, block ) | (fn proto extern, ";" );
